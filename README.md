# Доставка продуктов

Структуру функциональных возможностей (Mind Map)
```mermaid
mindmap
  root((Доставка продуктов))
    Заказ
      Оформление заказа
      Проверка наличия товаров
      Подтверждение заказа
    Пользователь
      Регистрация / Вход
      Личный кабинет
        История заказов
        Адреса доставки
        Способы оплаты
    Оплата
      Онлайн-оплата
      Платежный шлюз
      Подтверждение платежа
    Уведомления
      E-mail уведомления
      SMS уведомления
      Push-уведомления
    Доставка
      Планирование маршрута
      Отслеживание доставки
      Подтверждение доставки
```
